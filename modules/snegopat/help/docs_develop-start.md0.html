Это небольшая инструкция, которая поможет скриптописателям повысить удобство и скорость разработки скриптов для снегопата.

Предлагаем использовать для этих целей редактор Visual Studio Code и язык TypeScript. Этот язык - развитие языка JavaScript, к которому добавлена типизация переменных, классы и модули. Скрипт на языке TypeScript транслируется его компилятором в обычный JavaScript.

Благодаря возможности подключить к редактору файлы с описанием SnegopatAPI и типов 1С, при разработке скриптов для снегопата в редакторе отлично работает intellisense, что делает работу быстрой и комфортной.
