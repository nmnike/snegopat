    Просматривая различные настройки, хранящиеся в профайле 1С, наткнулся на настройку цветов для разных категорий слов в редакторе запросов. Для модулей подобные настройки вынесены в пользовательский интерфейс, а для запросов - нет. Этот скрипт исправляет данную несправедливость.
