Порядок развертывания снегопата в компании

Для развертывания снегопата для работы нескольких пользователей лучше придерживаться следующей схемы:

  • Установить основные компонеты центрального Снегопата, который будет синхронизироваться с snegopat.ru и раздавать изменения остальным клиентам.
  • Установить копии компонент снегопата конкретным разработчикам и настроить их синхронизацию с вашим центральным репозитарием.
